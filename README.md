## 📊 Sales by Region Analysis (Excel Project)

## 🔍 Project Objective

- To analyze and compare sales performance across different regions using a simple Excel bar chart

## 🛠 Tools Used
- Microsoft Excel

## 📁 Dataset
The dataset contains sales data for 4 regions:
- North – 120
- South – 150
- East – 90
- West – 110

## 📊 Visualization
## A bar chart was created to visually compare regional sales.

## 💡 Key Insights
- South region has the highest sales (150)
- East region has the lowest sales (90)
- North and West show moderate performance

- Data structuring in Excel
- Creating bar charts
- Understanding basic data visualization
- Extracting insights from simple datasets
